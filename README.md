Cashflow Tracker 

This is a simple Expense / Cashflow Tracking Application built using Flask and SQLite.

Features

Add daily expenses

Categorize expenses (Food, Travel, Shopping, Bills, Others)

View expense list

Local database (SQLite)


Tech Stack

Python

Flask

SQLite

HTML / Templates


How to Run Locally

1. Clone this repository


2. Open in VS Code


3. Run this command:



python app.py

4. Open browser:



http://127.0.0.1:5000

Database file: tracker.db
